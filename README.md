#### ***Prompts***

*Seção referente aos prompts aplicados sob o NotebookLM.*

| Assuntos | Prompts Sugeridos |
| :--- | :--- |
| ***O que é NotebookLM*** | Com base nos documentos fornecidos, elabore uma definição técnica e funcional do NotebookLM. Destaque sua arquitetura principal e como ele se diferencia de um chatbot de uso geral. |
| ***Como utilizá-lo com Gemini*** | Atue como um guia técnico. Analise a documentação e descreva o passo a passo para utilizar as capacidades do Gemini dentro do ambiente do NotebookLM, focando no processamento das fontes. |
| ***Quais são seus benefícios*** | Crie uma lista analítica dos principais benefícios citados nas fontes. Categorize-os em: Produtividade, Precisão (redução de alucinação) e Organização de Conhecimento. |
| ***Quais são os próximos passos*** | Considerando as funcionalidades da documentação, sugira um roteiro de implementação (roadmap). O que deve ser feito logo após o upload das fontes para extrair o máximo valor? |
